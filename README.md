## 1.
    var a = [];
    for (var i=0; i<10; i++) {
      a[i] = function(){console.log(i)}
    }
    a[6]();

 ## 解析 以上for循环代码可以解析为：
    (代码块1;代码块2；代码块3) { 语句}
    执行顺序： 代码块1(var了一个全局变量) => 代码块2 (符合条件) =>语句 => 代码块3(i自增) =>...(循环)=> 代码块2 (不符合条件) =>循环结束执行下一步 a[6]()
     (在代码块2的条件符合情况下循环调用,不符合条件跳出)
    代码块1 var() 了一个变量,此变量为全局变量 每次循环变量i自增,因此循环结束 i为全局变量 值为10
    因此a[6]()打印的结果为10
